PostgreSQL User Groups

The PostgreSQL community is proud to have many local chapters that advocate and educate users about PostgreSQL. Below is a list of PostgreSQL User Groups (PUGs) sorted by country and local area. If you would like to start a PostgreSQL User Group, please join the pgsql-advocacy mailing list and describe the PUG that you want to create.

If a PUG already exists in your area, follow the URLs below to find out how to attend and participate.
If you know of an international PostgreSQL community site that isn't listed on this page, please contact webmaster@postgresql.org.
